# PRODUCT VISION

## 1. Project Information

**Project Name**
D'Jaemo Jamur Krispi Web Store

**Project Type**
E-Commerce Website

**Version**
1.0 (MVP)

**Status**
Planning

---

## 2. Project Purpose

Membangun website e-commerce khusus untuk penjualan produk Jamur Krispi D'Jaemo secara online di Indonesia.

Website difokuskan pada kemudahan pelanggan dalam membeli produk, melakukan pembayaran melalui Midtrans, serta pengiriman menggunakan Biteship.

Seluruh pengelolaan website dilakukan oleh satu orang admin (owner) melalui Dashboard Admin.
## 3. Project Goals

Website ini memiliki tujuan utama sebagai berikut:

* Menjual produk Jamur Krispi D'Jaemo secara online.
* Mempermudah pelanggan melakukan pemesanan tanpa harus membuat akun.
* Menghitung ongkos kirim secara otomatis menggunakan Biteship.
* Memproses pembayaran secara aman menggunakan Midtrans.
* Memudahkan owner mengelola produk, pesanan, banner, dan pengaturan website melalui Dashboard Admin.
* Menyediakan sistem yang mudah dikembangkan untuk penambahan fitur di masa depan tanpa mengubah struktur utama aplikasi.
## 4. Project Scope (MVP)

Versi 1 (MVP) dari website hanya akan memiliki fitur-fitur berikut:

### Customer

* Melihat halaman beranda.
* Melihat daftar produk.
* Melihat detail produk.
* Menambahkan produk ke keranjang.
* Checkout sebagai Guest (tanpa login).
* Menghitung ongkos kirim menggunakan Biteship.
* Melakukan pembayaran menggunakan Midtrans.
* Melihat status pesanan.
* Menghubungi admin melalui halaman Contact.

### Admin

* Login ke Dashboard Admin.
* Melihat ringkasan penjualan.
* Mengelola produk.
* Mengelola pesanan.
* Mengelola banner dan promo.
* Mengelola pengaturan website.

Fitur di luar daftar ini tidak termasuk dalam versi pertama (MVP).
## 5. Out of Scope

Fitur-fitur berikut tidak akan dibuat pada versi pertama (MVP):

* Login Customer
* Registrasi Customer
* Wishlist
* Riwayat Pembelian
* Program Member / Loyalty Point
* COD (Cash On Delivery)
* Marketplace Integration (Shopee, Tokopedia, dll.)
* Mobile Application (Android / iOS)
* Multi Admin
* Multi Store
* Multi Language
* Pengiriman Internasional

Fitur-fitur di atas dapat dipertimbangkan untuk versi berikutnya apabila dibutuhkan.
