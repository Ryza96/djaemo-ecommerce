## 4. Admin Dashboard Pages

### Dashboard

Fungsi:

Menampilkan ringkasan kondisi toko.

Fitur:

- Total Penjualan
- Total Pesanan
- Pesanan Baru
- Produk Terjual
- Notifikasi

---

### Orders

Fungsi:

Mengelola seluruh pesanan customer.

Fitur:

- Daftar Pesanan
- Detail Pesanan
- Status Pesanan
- Detail Pembayaran
- Detail Pengiriman
- Cetak Resi
- Nomor Resi

---

### Products

Fungsi:

Mengelola seluruh produk.

Fitur:

- Tambah Produk
- Edit Produk
- Hapus Produk
- Upload Gambar
- Atur Harga
- Atur Stok
- Atur Status Produk

---

### Banner & Promo

Fungsi:

Mengelola tampilan halaman utama.

Fitur:

- Hero Banner
- Banner Promo
- Voucher
- Informasi Promo

---

### Settings

Fungsi:

Mengelola konfigurasi website.

Fitur:

- Informasi Toko
- Kontak
- Logo
- Media Sosial
- API Keys (melalui Environment, bukan ditampilkan)
