# SOFTWARE REQUIREMENT SPECIFICATION (SRS)

## 1. Introduction

### 1.1 Purpose

Dokumen ini mendefinisikan seluruh kebutuhan fungsional dan non-fungsional untuk website penjualan Jamur Krispi D'Jaemo.

Dokumen ini akan menjadi acuan utama dalam proses desain sistem, pengembangan aplikasi, pengujian, dan pemeliharaan proyek.

---

### 1.2 Project Overview

Website ini merupakan aplikasi e-commerce yang digunakan untuk menjual produk Jamur Krispi D'Jaemo secara online.

Sistem terdiri dari dua bagian utama:

* Website Customer
* Dashboard Admin

Website Customer digunakan untuk melakukan pembelian produk, sedangkan Dashboard Admin digunakan untuk mengelola seluruh aktivitas operasional website.

---

### 1.3 Actors

Sistem memiliki dua aktor utama.

#### Guest Customer

Hak akses:

* Melihat produk.
* Menambahkan produk ke keranjang.
* Checkout tanpa login.
* Melakukan pembayaran.
* Melihat status pesanan.
* Menghubungi admin.

#### Admin

Hak akses:

* Login Dashboard.
* Mengelola produk.
* Mengelola pesanan.
* Mengelola banner.
* Mengelola promo.
* Mengelola pengaturan website.
